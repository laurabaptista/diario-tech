# VS Code — Extensões Favoritas

- **Prettier** — formata o código automaticamente ao guardar
- **Live Server** — abre o HTML no browser em tempo real
- **HTML CSS Support** — autocomplete para HTML e CSS
- **Error Lens** — mostra erros diretamente na linha de código
- **GitLens** — visualiza o histórico Git dentro do VS Code
- **Material Icon Theme** — ícones mais bonitos para os ficheiros
