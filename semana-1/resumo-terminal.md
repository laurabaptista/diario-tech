# Terminal — 10 Comandos Úteis

1. `cd` — navegar entre pastas
2. `ls` — listar ficheiros e pastas
3. `mkdir` — criar uma nova pasta
4. `touch` — criar um novo ficheiro
5. `mv` — mover ou renomear ficheiros
6. `rm` — apagar ficheiros
7. `cp` — copiar ficheiros
8. `pwd` — mostrar o caminho atual
9. `clear` — limpar o terminal
10. `code .` — abrir a pasta no VS Code
